# bmcgrath

bmcgrath is my homepage mage using [React](https://react.dev/) + [Vite](https://vitejs.dev/)

## Installation

Use Node.js v18.14.1

```bash
npm install
```

## Serve locally

```bash
npm run dev
```

## Build

```bash
npm run build
```
